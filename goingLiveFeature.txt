## API Endpoints Implemented

### Auth
```
POST /api/auth/signup      - Register with email/password
POST /api/auth/login       - Login, get JWT
POST /api/auth/logout      - Invalidate token (client-side)
POST /api/auth/refresh     - Refresh JWT
GET  /api/auth/me          - Get current user
GET  /api/auth/google      - Get Google OAuth URL
POST /api/auth/google/callback - Complete OAuth flow
```

### Sessions
```
GET    /api/sessions           - List user's sessions
GET    /api/sessions/{id}      - Get session details
DELETE /api/sessions/{id}      - Delete session
GET    /api/sessions/{id}/transcript - Download transcript (text or JSON)
```

### Settings
```
GET    /api/settings/api-keys          - List stored keys (masked)
PUT    /api/settings/api-keys/{provider} - Store API key (validates first)
DELETE /api/settings/api-keys/{provider} - Remove API key
POST   /api/settings/api-keys/{provider}/validate - Validate stored key
```

### Billing
```
GET  /api/billing/subscription  - Current tier and usage stats
GET  /api/billing/usage         - Usage history
POST /api/billing/payment       - Process payment (mock)
GET  /api/billing/payments      - Payment history
GET  /api/billing/can-start-session - Check if user can start session
```

## How to Start

1. Checkout branch: `git checkout ralph/interview-assistant-production`
2. Install backend dependencies: `cd backend && pip install -r requirements.txt`
3. Apply migrations: `cd backend && python -m alembic upgrade head`
4. Generate encryption key and update .env:
   ```python
   python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
   ```
5. Start backend: `cd backend && uvicorn main:app --reload`
6. Install frontend dependencies: `cd frontend && npm install`
7. Start frontend: `cd frontend && npm run dev`
8. Visit http://localhost:3000
